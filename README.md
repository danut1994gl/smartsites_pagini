# SmartSites - Pagini Servicii Dezvoltare Software

## Structura Paginilor

### Pagini Principale (Servicii)
1. `01-software-custom.md` - Soluții Software Custom pentru Companii
2. `02-aplicatii-mobile.md` - Dezvoltare Aplicații Mobile Android & iOS
3. `03-magazin-online-app.md` - Aplicație Mobilă pentru Magazin Online (WooCommerce)
4. `04-mvp-lansare-rapida.md` - Lansare Rapidă de Platforme și MVP-uri

### Pagini Suport (Încredere & Conversie)
5. `05-despre-noi.md` - Despre Noi
6. `06-studii-de-caz.md` - Studii de Caz
7. `07-proces-colaborare.md` - Procesul de Colaborare
8. `08-suport-dezvoltare.md` - Suport și Dezvoltare Ulterioară
9. `09-faq.md` - Întrebări Frecvente
10. `10-contact.md` - Contact / Discuție Proiect

### Fișiere Auxiliare
- `seo-meta.md` - Meta titles + descriptions pentru toate paginile

## Ton & Stil
- **Antreprenorial, dar matur** - clar, direct, orientat pe rezultate
- **Fără jargon tehnic** în prima jumătate a paginilor
- **Fără prețuri** - estimare după analiza cerințelor
- **Fără menționarea partenerului tehnic** - totul apare ca serviciu SmartSites
